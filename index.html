The Governance Gap

CEOs are expected to lead systemic transformations that take 7 to 10 years to mature. Boards judge performance before those transformations materialise. Average CEO tenure has fallen to 7.1 years. An increasing number of mandates are compressed to 3 to 3.5 years. Total Shareholder Return tells the board how the market responded. It says nothing about whether the firm built or eroded the asset base that will generate future returns.

A measurement gap creates a governance gap. Without lenses that make asset preservation visible, boards and reward committees lack the tools to incentivise decisions that secure the firm's long-term viability.

Extended Shareholder Return

Extended Shareholder Return addresses this gap. It retains Total Shareholder Return as the external reference for realised performance, while introducing asset integrity as a governing condition. Two lenses help boards see whether current returns come from preserving or drawing down the asset base. The first adds an Asset Integrity Multiplier to TSR. The second tracks system restoration whenever extraction is material. The framework ties CEO compensation to whether the asset base was preserved, and creates space for transformation decisions that extend beyond any single tenure.
